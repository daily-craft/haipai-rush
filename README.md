# HAIPAI RUSH

配牌14枚で役満が出るまでひたすらシミュレートする麻雀配牌シミュレーター。

**https://daily-craft.github.io/haipai-rush/**

## 機能

- 配牌14枚をランダムに生成し、役満成立をチェック
- 10種の役満に対応（天和、国士無双、九蓮宝燈、四暗刻、大三元、小四喜、大四喜、字一色、清老頭、緑一色）
- 狙う役満を自由に選択可能
- 速度調整（×1〜×10,000）
- 2Dモード / 3Dモード（Three.js）

## 技術スタック

- HTML / CSS / JavaScript（フレームワークなし）
- Three.js r158（3Dモード）
- GitHub Pages（ホスティング）
